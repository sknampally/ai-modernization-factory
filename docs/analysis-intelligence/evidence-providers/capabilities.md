# Capabilities

Stable hierarchical IDs include:

- `source.files`
- `dependencies.imports`
- `dependencies.type-only`
- `dependencies.runtime`
- `architecture.units`
- `architecture.layers`
- `architecture.composition-roots`
- `architecture.registration`
- `framework.usage`
- `framework.annotations`
- `tests.presence`

Maturity: `experimental`, `partial`, `supported`, `mature`.

Unsupported capabilities are declared explicitly and must not be treated as
absent evidence.
