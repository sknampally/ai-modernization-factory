# Coverage Presentation

Coverage states must remain accurate:

| Status | Presentation |
| ------ | ------------ |
| measured | Measured percentage / complete wording |
| partial | Partial with ratio when available |
| unsupported | Not measured by current provider set (not 0%) |
| not_applicable | Not applicable (enterprise context absent, etc.) |
| unknown | Unavailable (not 0%) |

Areas include extraction, classification, architectural units, dependencies,
framework, build metadata, and enterprise context.
