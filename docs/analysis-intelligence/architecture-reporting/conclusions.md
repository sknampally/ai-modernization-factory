# Conclusions Presentation

Conclusions are the primary executive layer.

Shown:

- title, category, summary
- materiality, confidence
- affected scope
- severity summary
- primary / supporting findings
- modernization relevance (wave)
- business impact (`Unknown` / `Not assessed` / `Provided enterprise context`)
- recommendation group links
- conclusion-specific limitations

Ordering follows assessment/policy order. Policy IDs stay in technical
traceability, not as titles.
