# CLI

Inspect presentation content from `report.json` (no re-analysis):

```bash
aimf architecture report inspect --report path/to/report.json
aimf architecture report conclusions --report path/to/report.json
aimf architecture report recommendations --report path/to/report.json
aimf architecture report findings --report path/to/report.json
aimf architecture report coverage --report path/to/report.json
aimf architecture report limitations --report path/to/report.json
aimf architecture report traceability --report path/to/report.json
```

All commands support `--json`.

Assessment-layer CLI remains under `aimf architecture assessment …`.
