# Architecture Conclusions

Phase **4.2.3** — Architecture Conclusions and Aggregation.

Conclusions interpret architecture findings without replacing them.

| Concept | Meaning |
| ------- | ------- |
| Rule finding | A specific observed condition |
| Finding relationship | Deterministic connection between observations |
| Architecture conclusion | Bounded interpretation of one or more observations |
| Consolidated recommendation | Coordinated response to related observations |
| Report statement | Presentation of a conclusion for an audience |

Disabled by default: `[analysis.architecture_conclusions] enabled = false`.
