# Domain Model

`ArchitectureConclusion` includes deterministic `conclusion_id`, category, title,
technical/executive interpretations, source finding IDs, severity summary,
confidence, coverage, materiality, business impact (`unknown` for repository-only),
modernization wave, limitations, and status.

Related: `FindingRelationship`, `FindingCluster`, `ConsolidatedRecommendation`,
`ArchitectureSummary`.
