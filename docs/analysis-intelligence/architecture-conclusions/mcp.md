# MCP

Generic architecture-conclusion tools (no per-policy tools; no source leakage):

- `list_architecture_conclusion_policies`
- `inspect_architecture_conclusion_policy`
- `list_architecture_conclusions` — requires a persisted `architecture_conclusions.json` artifact
- `inspect_architecture_conclusion`
- `explain_architecture_conclusion` — source findings, relationships, consolidated recommendations, coverage, and limitations

Enterprise-only context remains optional and is not required for policy listing.
