# CLI

```text
aimf architecture assessment inspect --artifact …/architecture-assessment.json
aimf architecture assessment findings --artifact …
aimf architecture assessment conclusions --artifact …
aimf architecture assessment recommendations --artifact …
aimf architecture assessment coverage --artifact …
aimf architecture assessment limitations --artifact …
aimf architecture assessment traceability --artifact …
```

`--json` is supported. The assess command remains `aimf assess`.
